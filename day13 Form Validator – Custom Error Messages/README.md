# Form Validator

A simple and modern form validation project using HTML, CSS, and JavaScript.  
It validates user inputs with custom error messages and clean UI.

---

## Features

- Required field validation
- Custom error messages
- Email format verification
- Password length validation
- Confirm password match check
- Responsive and clean design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## How It Works

1. User fills out the registration form.
2. JavaScript checks:
   - If fields are empty
   - If email format is valid
   - If password meets the minimum length
   - If passwords match
3. Errors are shown under each input if validation fails.
4. When all inputs are valid, the form can be submitted.
